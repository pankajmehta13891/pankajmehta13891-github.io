Iris Dataset Mini Dashboard

This project presents a Mini Dashboard for the Iris Dataset, designed to demonstrate data visualization, analytics, and machine learning concepts using the Iris flower dataset. The dashboard provides an interactive environment to explore dataset features, visualize relationships, and understand classification behavior.

The Iris dataset is widely used in data science and machine learning for classification tasks. It contains measurements of sepal length, sepal width, petal length, and petal width for three species — Setosa, Versicolor, and Virginica. This project highlights petal-based visualization for better classification insights.

Dashboard Components and Purpose

  • Local CSV Canvas Dashboard — Visualizes Iris data from a local iris.csv file.
  • URL Fetch Dashboard — Retrieves dataset data from URL.
  • Iframe Dashboard — Displays external visualization files.
  • Colab Notebook Dashboard — Opens analysis and ML notebooks.
  • Dataset Analytics Section — Shows statistical insights.

Application Architecture

Each dashboard component is implemented independently with separate CSS and JavaScript files to ensure modularity, maintainability, and scalability.
